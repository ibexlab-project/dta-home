# Media Room Images

This directory contains images for the Media Room section.

## Expected Files

- `media-mou-signing.png` - MOU signing ceremony image with Tourism Authority of Thailand
- `media-simx-launch.png` - SiMX launch related image
- `media-pilot-trust.png` - Pilot trust related image

## Requirements

- Format: PNG
- Size: Minimum 400x300px (maintain aspect ratio)
- Description:
  1. `media-mou-signing.png`: MOU signing ceremony image
  2. `media-simx-launch.png`: SiMX launch related image (or background image)
  3. `media-pilot-trust.png`: Pilot trust related image (or background image)
